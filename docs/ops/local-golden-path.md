# Local Golden Path

1. `nix develop` or install python deps locally.
2. Run `just bootstrap`.
3. Run `just run`.
4. Open `http://127.0.0.1:8000/ui`.
5. Load Docs Pack from GUI.
6. Build an agent bundle from Agent Builder panel.
7. Publish custom workflow in Workflow Lifecycle panel.
8. Intake and run task from Task Run Console.
9. Inspect trace/events and topology in GUI.
